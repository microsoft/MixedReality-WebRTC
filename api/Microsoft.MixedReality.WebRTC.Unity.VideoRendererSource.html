    Class VideoRendererSource

    Base class for video-producing entities which can be rendered using a VideoRenderer.

    This class is only used because Unity requires a common base class between VideoTrackSource and VideoReceiver to be able to serialize a polymorphic property to either of those producers in VideoRenderer. Ideally VideoRenderer would use an IVideoSource property instead, but polymorphic serialization of interfaces is not supported (there is partial support in Unity 2019.3 but some things like the component picking window are still broken).
